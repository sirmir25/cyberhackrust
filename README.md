## 🌟 What is CyberHack?

**CyberHack** is an immersive terminal-based hacking simulator that puts you in the shoes of an elite cyber warrior. Experience the thrill of infiltrating corporate networks, battling AI defense systems, and uncovering conspiracies in a cyberpunk world where information is power and anonymity is survival.

### 🎭 The Story

> *"In 2025, the world is controlled by mega-corporations and their AI surveillance networks. You are a member of an underground resistance fighting for digital freedom. Your weapon? A terminal. Your battlefield? The digital realm."*

## 🎮 Quick Start

```bash
# Clone the repository
git clone https://github.com/your-username/cyberhack.git
cd cyberhack

# Build and run
cargo build --release
cargo run

# Optional: Install globally
cargo install --path .
cyberhack
```

### 🎯 First Steps
1. **Create your hacker alias** - Choose your handle wisely
2. **Complete the tutorial** - Learn basic commands
3. **Start your first mission** - Infiltrate Evil Corp
4. **Upgrade your skills** - Become the ultimate hacker

## 🛠️ Commands Arsenal

<details>
<summary>🔍 <b>Reconnaissance & Scanning</b></summary>

| Command | Description | Example |
|---------|-------------|---------|
| `scan` | Discover networks and systems | `scan 192.168.1.0/24` |
| `nmap` | Advanced port scanning | `nmap -sS target.com` |
| `whois` | Domain information lookup | `whois evil-corp.com` |
| `netstat` | Show network connections | `netstat -an` |

</details>

<details>
<summary>⚡ <b>Infiltration & Exploitation</b></summary>

| Command | Description | Example |
|---------|-------------|---------|
| `connect` | Connect to remote system | `connect 10.0.0.1` |
| `exploit` | Execute vulnerability exploit | `exploit buffer_overflow` |
| `crack` | Crack passwords | `crack -w wordlist.txt` |
| `backdoor` | Install persistent access | `backdoor install` |

</details>

<details>
<summary>🛡️ <b>Stealth & Anonymity</b></summary>

| Command | Description | Example |
|---------|-------------|---------|
| `proxy` | Route through proxy | `proxy socks5://127.0.0.1:9050` |
| `tor` | Enable Tor routing | `tor enable` |
| `vpn` | Connect to VPN | `vpn connect sweden` |
| `spoof` | Spoof MAC address | `spoof -m random` |

</details>

<details>
<summary>💻 <b>System Control</b></summary>

| Command | Description | Example |
|---------|-------------|---------|
| `ls` / `dir` | List directory contents | `ls -la /home/user` |
| `cat` | Display file contents | `cat secret.txt` |
| `download` | Download files | `download database.sql` |
| `upload` | Upload files | `upload malware.exe` |

</details>

<details>
<summary>🎯 <b>Special Operations</b></summary>

### Prerequisites
- **Rust** 1.70+ ([Install Rust](https://rustup.rs/))
- **Terminal** with Unicode support
- **Audio** (optional, for sound effects)

### 📦 Installation Methods

#### Option 1: From Source
```bash
git clone https://github.com/your-username/cyberhack.git
cd cyberhack
cargo build --release
./target/release/cyberhack
```

#### Option 2: Cargo Install
```bash
cargo install cyberhack

```
